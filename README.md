# EcoBazaarX

EcoBazaarX is a web application for eco-friendly shopping that includes secure authentication, a user dashboard, cart and wishlist features, and order management.

## Tech Stack
- React
- TypeScript
- Vite
- React Router DOM
- Tailwind CSS
- Recharts

## Features
- User login and signup
- Protected routes
- Dashboard
- Cart and wishlist
- Order history
- Profile management

## Setup
```bash
npm install
npm run dev
Build
npm run build
```
